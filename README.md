# CPRD MedCode IDs for Long-Term Conditions and Fragility Fracture Outcomes

## Overview

This repository contains the Clinical Practice Research Datalink (CPRD) MedCode IDs used to identify long-term conditions (LTCs) and fragility fracture outcomes in the study:


The code lists are provided to support transparency, reproducibility, and reuse of the research methods.

## Repository Contents

This repository includes CPRD MedCode IDs for the following conditions and outcomes:

### Long-Term Conditions

- Type 1 Diabetes (T1D)
- Type 2 Diabetes (T2D)
- Maturity-Onset Diabetes of the Young (MODY)
- Sleep Apnoea
- Asthma
- Cancer
- Chronic Kidney Disease (CKD)
- Chronic Obstructive Pulmonary Disease (COPD)
- Cardiovascular Disease (CVD)
- Dementia
- Depression
- Hypertension
- Liver Disease
- Nephropathy
- Neuropathy
- Osteoarthritis
- Osteoporosis
- Retinopathy
- Rheumatological Conditions
- Falls

### Fragility Fracture Outcomes
- Hip Fracture
- Pelvic Fracture
- Proximal Humerus Fracture
- Radius Fracture
- Vertebral Fracture

Each worksheet contains the CPRD MedCode IDs and corresponding clinical terms used to identify these conditions and outcomes within the CPRD.

## Data Source

The MedCode IDs were developed using the CPRD clinical terminology and were used to identify diagnoses recorded in primary care.

Researchers using these code lists should ensure that they are appropriate for their CPRD version and study design.
